# Pen Tool Flyout Panel Extension for Photoshop

Created to add functionality from Adobe Illustrator into Photoshop

## Build

### Dependencies

* [node](https://nodejs.org/en/download/)
* [gulp](https://gulpjs.com/)

`npm install`

`gulp`
